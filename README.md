# BuddyRoute 🗺️

An offline-first travel journal and GPS route tracking app for Android.

## Features
- Live GPS route tracking with Google Maps
- Trip journaling with photos, voice notes, and mood tags
- Auto-generated PDF trip stories with themed designs
- Achievement system, travel streaks, and personality insights
- Emergency SOS with shake detection and SMS alerts
- Trip comparison and "Where was I?" calendar view

## Tech Stack
Java • MVVM • Room Database • Google Maps SDK • Glide • 
Canvas/PdfDocument • Material Components

## Setup
1. Clone the repo
2. Add your own Google Maps API key in `AndroidManifest.xml`
3. Build and run in Android Studio (min SDK 26)
